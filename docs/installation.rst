Installation
============

Option 1 (recommended): install pre-compiled package from conda-forge
---------------------------------------------------------------------

Install the conda-forge package ``libparallelproj``:

.. tab-set::

	.. tab-item:: mamba

		.. code-block:: bash

			mamba install -c conda-forge libparallelproj

	.. tab-item:: conda

		.. code-block:: bash

			conda install -c conda-forge libparallelproj


Option 2: compile from source
-----------------------------

For source builds, follow the instructions in the `GitHub project README <https://github.com/KUL-recon-lab/libparallelproj/blob/main/README.md>`_
